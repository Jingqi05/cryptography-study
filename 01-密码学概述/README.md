# 密码学概述

## 什么是密码学

密码学（Cryptography）是研究如何在不安全环境中安全通信的学科，核心目标是保护信息的**机密性**、**完整性**、**认证性**和**不可否认性**。

## 核心目标（CIA+）

| 目标 | 说明 | 实现手段 |
|------|------|---------|
| **机密性** (Confidentiality) | 只有授权方能读取信息 | 对称加密、非对称加密 |
| **完整性** (Integrity) | 信息未被篡改 | 哈希算法、MAC、数字签名 |
| **认证性** (Authentication) | 确认通信方身份 | 数字证书、数字签名 |
| **不可否认性** (Non-repudiation) | 发送方无法否认已发送 | 数字签名 |

## 发展历史

```
古典密码          近代密码           现代密码              后量子密码
(公元前~1900)    (1900~1970)      (1970~至今)           (2024~)
    │                │                 │                    │
    ▼                ▼                 ▼                    ▼
  凯撒密码        Enigma机         Diffie-Hellman        NIST PQC
  维吉尼亚密码    DES标准          RSA公钥体制           ML-KEM/ML-DSA
  换位密码        对称加密成熟      AES/ECC/SHA           抗量子算法
```

### 里程碑事件

| 年份 | 事件 |
|------|------|
| ~50 BC | 凯撒密码（替换密码） |
| 1918 | Enigma 密码机诞生 |
| 1976 | Diffie-Hellman 密钥交换协议 |
| 1977 | RSA 算法提出 |
| 2001 | AES 标准发布（取代 DES） |
| 2010 | SM2/SM3/SM4 国密标准发布 |
| 2015 | TLS 1.3 草案 |
| 2024 | NIST 发布首批后量子密码标准 |

## 密码学分类体系

```
密码学
├── 对称加密（Symmetric）
│   ├── 分组密码：AES, DES, SM4
│   └── 流密码：ChaCha20, RC4
├── 非对称加密（Asymmetric）
│   ├── 基于大数分解：RSA
│   ├── 基于离散对数：DH, DSA
│   └── 基于椭圆曲线：ECC, ECDH, ECDSA, SM2
├── 哈希函数（Hash）
│   ├── SHA 家族：SHA-1, SHA-256, SHA-3
│   ├── 国密：SM3
│   └── 新兴：BLAKE3
└── 密码学协议
    ├── TLS/SSL
    ├── SSH
    ├── IPsec
    └── PKI/X.509
```

## 三者协作关系

以 TLS 1.3 握手为例：

```
客户端                                服务器
  │                                    │
  │── ClientHello ────────────────────>│
  │<── ServerHello + 证书 ─────────────│
  │                                    │
  │   【非对称加密】验证身份、协商密钥     │
  │   ECDHE 密钥交换 + ECDSA 签名验证   │
  │                                    │
  │── 密钥交换完成 ───────────────────>│
  │                                    │
  │   【对称加密】加密后续通信数据        │
  │   AES-256-GCM                      │
  │                                    │
  │   【哈希算法】完整性校验             │
  │   SHA-384                          │
  │                                    │
```

> **核心原则**：非对称解决信任和密钥分发，对称解决高效数据加密，哈希解决完整性验证。三者协同构成完整安全通信体系。
